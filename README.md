# NextWave Innovations 🌐

A modern, responsive website showcasing **NextWave Innovations** — a tech company focused on web development, app design, and cybersecurity.

## 🚀 Features
- Smooth navigation bar with links to Home, About, Services, Portfolio, and Contact.
- Responsive design using Flexbox and Grid.
- Dark/Light mode toggle button for a modern user experience.
- Portfolio gallery with hover effects.
- Contact form with newsletter subscription.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, transitions)
- JavaScript (Dark mode toggle)

## 📸 Sections
- **Home**: Hero banner with tagline.
- **About**: Company overview.
- **Services**: Cards for Web Development, App Design, Cybersecurity.
- **Portfolio**: Project gallery with hover effects.
- **Contact**: Email and subscription form.
